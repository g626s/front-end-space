Mono Clock

Last updated: 10.8.2024

Displays a interactive digital clock and allows user interaction to show their current time. Here's a quick breakdown of the content and functionality:

  • Retrieve the current time (hours and minutes)
  • Update the clock display in the DOM to reflect the local time
  • Display a message below the button indicating the user's geographic location (region and city) based on their timezone
  • Push event details (like time, region, and city) to the dataLayer for tracking

URL Parser

Last updated: 10.10.2024

The URL Parser Tool is a web-based application designed to analyze and extract components from user-provided URLs. It enables users to enter a URL and receive detailed information about its structure, including protocols, hosts, paths, and query parameters. The tool aims to simplify the understanding of URL components and enhance users' ability to debug and manage URLs effectively:

  • Users can input a URL into a designated input field
  • URL Parsing
  • Event tracking

Cookie Banner (Testing Only)

Last updated: 11.6.2024

This cookie banner is designed purely as a testing demonstration and is not intended for use as a functional or legally compliant cookie management tool. User consent decisions are logged in the dataLayer to mimic how consent states might be tracked in analytics setups. Users can open a "Manage Preferences" dialog to select or deselect cookie categories. The categories include: Essential (cannot be disabled), Targeting (adds GSLC5), Personalization (adds GSLC4), Analytics (adds GSLC3). After submission, cookie_active_groups updates in the dataLayer based on selected categories.

Key Features:

  • Default Consent Group
  • DataLayer Integration
  • Preference Management

Disclaimer: This cookie banner project is for testing purposes only and is not intended to fulfill legal compliance requirements. It does not manage real cookies and does not provide full functionality as would a production-ready cookie consent solution.
